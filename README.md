**蔚蓝档案汉化工具集** —— 在不触及敏感内容的情况下，开源部分核心逻辑，为 BA 本地化汉化提供技术支持。

---

## 项目简介

本项目旨在为 *Blue Archive*（JP, GL）提供高效的汉化制作支持。
